# 位置优化器

本模块专注于优化模具布局中各组件的位置分布，通过评估相对位置和整体布局来优化空间平衡性。

## 关键指标

1. 相对位置评估
   - 计算组件间的位置关系
   - 评估位置分布的合理性
   - 确保布局的整体性能

2. 布局平衡分析
   - 分析组件的空间分布
   - 评估位置的协调性
   - 优化整体布局效果

## 实现指南

1. 核心配置（`core/config.ts`）
   - 定义位置评估参数
   - 设置布局约束条件
   - 配置优化权重系统

2. 评分系统（`scoring/calculator.ts`）
   - 实现位置评分计算
   - 评估布局平衡性
   - 计算综合布局得分

3. 优化策略（`optimizer.ts`）
   - 优化组件相对位置
   - 平衡整体布局效果
   - 考虑实际加工约束
